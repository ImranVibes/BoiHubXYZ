## 🌟 Core Platform Features

### 📚 Bangla Book Library & Smart Search (ফ্রি বাংলা বই ও উপন্যাস)
- **Wide Catalog:** Read and download popular Bangla novels, translated books, horror stories, Islamic books, poetry, and academic PDFs.
- **Instant Search:** Auto-suggest search algorithm that helps you find book names, author profiles, and category listings in milliseconds.
- **Mobile-First Design:** Swipeable category chips and subcategories designed for responsive touch screens.
- **Advanced Sorting:** Filter the book grid dynamically by "Most Popular" (download count) and "Newest" arrivals.

### ⚡ Seamless E-Book Reading & Downloading
- **High-Speed Cloud Storage:** Direct, high-speed PDF download links optimized for speed and reliability.
- **Adaptive Thumbnails:** Fluid multi-image cover sliders with high-fidelity WebP compression powered by Cloudflare CDN.
- **User-friendly Layout:** Auto-collapsing description blocks ("বিস্তারিত পড়ুন") for easy reading flow.

### 💬 Reader Engagement & Request Feed (বইয়ের অনুরোধ ফোরাম)
- **Interactive Requests:** Post requests for missing books, upload reference screenshots, and let other members of the community upload them.
- **Dynamic Comments:** Active discussion sections under books and requests, utilizing auto-growing textareas for detailed discussions.
- **Earn Rewards:** Participate in the community to earn XP, rank up on the global Leaderboard, and earn virtual coins to redeem premium benefits.

## 🛠️ Tech Stack & E-E-A-T Compliance

BoiHub is engineered with modern web technologies that prioritize performance, security, and search engine discoverability.

- **Frontend:** Next.js (App Router, React 19) utilizing Semantic HTML structure for optimal search crawler parsing.
- **Styling:** TailwindCSS v4 with absolute safe-area notch layout adjustments (`pb-safe`).
- **Database & Data Layer:** PostgreSQL database mapped with Prisma ORM ensuring strict access controls.
- **Caching Layer:** Redis (via `ioredis`) for blazing-fast database queries, session caching, and performance optimization.
- **Media Delivery:** Cloudflare R2 WebP CDN compression for fast page load times (improving Core Web Vitals SEO scores).
- **SEO Optimization:** Dynamic Canonical URLs, structured JSON-LD Schema integration (WebSite, Organization, Book schemas), and custom OpenGraph/Twitter card tags dynamically served on server-side nodes.
